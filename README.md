# 📄 Resume Analyzer (NLP)

## Overview

This project analyzes resume text using basic NLP techniques to extract skills and compare them with different job roles. It helps identify suitable roles and highlights missing skills.

## What the project does

* Cleans and processes resume text
* Extracts important skills from the resume
* Matches the extracted skills with predefined job roles
* Calculates a match score for each role
* Identifies missing skills (skill gap analysis)

## Example Output

* Match score for each job role
* Best suited job recommendation
* Missing skills required for improvement

## Files Included

* `resume_analyzer.ipynb` – main notebook
* `sample_resume.txt` – example resume input
* `requirements.txt` – required libraries

## Tools Used

* Python
* spaCy
